# How to Run the Code

Follow these three simple steps to get project up and running. 

Step 1: Create the Virtual Environment
Open  terminal in the project's root folder and run:
  - macOS/Linux: python -m venv venv

Step 2: Activate the Environment & Install Dependencies
Activate the virtual environment and install your requirements:
  - macOS/Linux: 
  - source venv/bin/activate
  - pip install -r requirements.txt

Step 3: Run the Code
Execute the main script located inside the src directory:
  - python src/main.py
